Sportcasts

Find out when and where your favorite team is playing. On any device.
Ruby Rails React NodeJS
Database + API
Supplying the data to all of the clients is a Rails server which scrapes NBC Sports for all new game information. Clients talk to the server via a typical REST API.

Web
For those who want a web expierence can use Sportcasts Web. It utilizes React to provide a quick, seamless experience for users who want to search for their team. Also supported are user profiles for users who want to take advantage of notifications. After signup, users can select their favorite teams and then configure notifications to receive when those teams are about to play. Currently supported are SMS notifications, with email coming soon.

Alexa Skill
The original client, Sportcasts Alexa provides a voice controlled interface for finding game information. To get game information, simply ask "Alexa, ask Sportcasts when the cavaliers play next" or "Alexa, ask Sportcasts what channel the Browns are on right now". At the time of development, Amazon did not support passing along location information - this made it difficult for Alexa to tell them the time in their specific timezone. As a result, support was added for users to tell Sportcasts what their zipcode was to ensure they got told the correct time.

Chrome Extension
Sportcasts Chrome is a simple extension that when clicked on in the toolbar, a little box pops up with a team select. Selecting a team will fetch the game information for that team's next known game.

Telegram Bot
Sportcasts Telegram is a bot that can be called via its alias "@sportcasts" along with a team name, and it will automatically respond with information on that team's next game, just like the Chrome extension.
