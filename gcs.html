Description
Bash
Git-cashier is a command line tool that shows the user their most recently modified git branches in their current project along with a helpful selection menu. Makes it easy to switch between branches without needing to remember or type long branch names.

Features include:
  • Works from any git project
  • Installable via Brew on MacOS
  • Allows for configuration of the number of recent branches to show
