Description
Ruby Rails
Spotify Time Machine is a web app that lets you view past versions of most popular Spotify playlists. Use it to view "Today's Top Hits" from last year or "New Music Friday" from last week.

Features include:
  • Playlist searching
  • Playlists in the system gain new version every time the playlist on Spotify is updated
  • Connect to Spotify to save favorite playlist versions directly to a user account
